== README

Simple Ruby on Rails Application that calculates the Body Mass Index. The application has a login feature used to register users in order to be able to calculate their BMI. Almost all the Front-End is build using React.

Languages/Tools/Frameworks Used:

- Ruby 2.5.0
- Rails 4.2.11
- React
- Sass
- Coffee
- Bootstrap
- Devise
- Cucumber Test (WIP)
